# Plugin Images

This directory contains images used by the plugin repository.

## Required Images

### logo.png
- **Size**: 512x512px recommended
- **Format**: PNG with transparency
- **Purpose**: Plugin icon shown in Jellyfin's plugin catalog
- **Content**: TheSportsDB or Formula 1 themed logo

## Creating a Logo

You can create a logo that combines:
- Formula 1 branding elements
- TheSportsDB branding
- Jellyfin styling

Make sure you have rights to use any logos or branding.

## Placeholder

Until a proper logo is created, you can use a simple text-based logo or the Formula 1 logo (if permitted).
